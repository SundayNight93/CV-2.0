Mis Habilidades.

Electronica

Más de dos años en el diseño, fabricación, mantenimiento y reparación de hardware para el sector gasolinero. Manejo de software de diseño como solidworks y eagle para la fabricación de PCB's, prototipos de dispositivos electrónicos, hasta la visualización e implementación de una instalación electrica y red para una oficina. Conocimiento del los lenguajes c, c++ y c# para la elaboración de interfaces que controlen distintos dispositivos electronicos mediante el uso de la red (IoT) y de igual manera para la programación de los microcontroladores de dichos circuitos.

Programación Web

Conocimiento de las tecnologias que se usan actualmente como son: html, css, javascript, nodejs, react, angular, mysql y python. Con la ayuda de éstas he podido realizar paginas web y aplicaciones mobiles orientadas a IoT.
